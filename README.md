# ibaciu6.github.io

Personal GitHub Pages dashboard — repo overview, RSS feed health, and links.

Lightweight single-page dashboard built with vanilla HTML/CSS/JS. Fetches live stats from the [rss-generator](https://github.com/ibaciu6/rss-generator) project to display feed health.

**Features**
- Repo cards with language badges and recency
- Activity heatmap (6-month timeline)
- Language breakdown bar + legend
- RSS feed health ticker (fetched live from rss-generator GH Pages)
- Quick links to GitHub profile, repos, RSS generator, OPML download

**Deployment**
Push to `main` — GitHub Pages auto-deploys to `ibaciu6.github.io`.

**Built with**
Vanilla HTML, CSS (dark theme, monospace), vanilla JS — no frameworks, no build step.
